# User-Facing Service Design

## Research Summary

Based on analysis of leading GPU cloud providers:

| Platform | Model | Strength | Weakness |
|----------|-------|----------|----------|
| **RunPod** | GPU Pods + Serverless | Templates, FlashBoot, Community Cloud | Complex pricing tiers |
| **Vast.ai** | Marketplace | Cheapest prices, flexibility | Variable reliability |
| **Lambda Labs** | On-Demand VMs | Enterprise-grade, simple | Limited availability |
| **Replicate** | Model API | One-line deployment, per-prediction pricing | No GPU access |
| **Together.ai** | Inference API | Fast, OpenAI-compatible | Token-based pricing |
| **Modal** | Serverless Code | Developer experience, sub-second cold starts | Learning curve |

**Sources:** [RunPod](https://docs.runpod.io/get-started), [Vast.ai](https://vast.ai/pricing), [Lambda Labs](https://lambda.ai/instances), [Replicate](https://replicate.com/pricing), [Together.ai](https://www.together.ai/pricing), [Modal](https://modal.com/pricing)

## User Personas

### 1. AI Developer (Primary - 60%)
- **Who**: Solo developers, small teams building AI products
- **Pain**: Token-based pricing is unpredictable, need flexibility
- **Want**: Easy deployment, predictable costs, API access
- **Budget**: $50-500/month

### 2. ML Researcher (20%)
- **Who**: Academic researchers, R&D teams
- **Pain**: Need raw GPU access for training, limited budgets
- **Want**: Cheap GPUs, Jupyter access, spot instances
- **Budget**: $100-2000/month

### 3. AI Startup (15%)
- **Who**: Funded startups building AI-first products
- **Pain**: Need to scale quickly, control costs
- **Want**: Auto-scaling, monitoring, team management
- **Budget**: $1000-10000/month

### 4. GPU Provider (5%)
- **Who**: Crypto miners, small data centers
- **Pain**: Idle GPUs not earning
- **Want**: Passive income, easy setup, reliable payouts
- **Revenue**: Earn 80-90% of GPU rental

## Key Differentiators

### 1. Hybrid Pricing Model
- **Model catalog**: Per GPU hour (predictable)
- **Custom pods**: Per GPU hour (flexible)
- **No token counting** - Use as much as GPU allows

### 2. Unified Experience
- One dashboard for models AND custom pods
- Same billing system
- Same API key structure

### 3. Community + Enterprise GPUs
- Community Cloud: Cheapest prices (marketplace)
- Secure Cloud: Enterprise reliability

### 4. OpenAI Compatibility
- Drop-in replacement for OpenAI SDK
- Same API structure
- Easy migration

### 5. Real-Time Voice Support
- WebSocket endpoints for streaming
- Voice agent pipeline (STT + LLM + TTS)
- Sub-second latency
